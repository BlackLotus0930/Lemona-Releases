# Changelog

All notable changes to Lemona will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-23

### Added
- Initial release of Lemona
- Rich text editor with formatting support
- WorldLab visual workspace
- Context-aware AI assistant
- Intelligent autocomplete
- Semantic search across documents
- Professional PDF and DOCX export
- Auto-update functionality
- Zen Mode (F11) for distraction-free writing
- Support for Windows, macOS, and Linux

### Features
- Multi-threaded chat interface
- Document version history
- Project-based organization
- PDF and DOCX file support
- Local data storage
- Offline capability
